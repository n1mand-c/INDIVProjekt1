# Бот “Фінансовий трекер студента” 

# Мета проєкту 
Розробити консольного бота, який допомагає студенту стежити за особистими витратами та бюджетом. Програма веде облік витрат, сортує їх за категоріями та автоматично обчислює залишок бюджету.



#Опис логіки та алгоритму роботи 
1. *Ініціалізація*: При запуску програма намагається зчитати дані з файлу `finance_data.json`. Якщо файлу немає, створюється базова структура (бюджет = 0, порожній список витрат).
2. *Взаємодія*: Бот очікує текстові команди від користувача (наприклад, "додати витрату", "залишок", "звіт за категоріями").
3. **Обробка та Контроль**:
   - При додаванні нової витрати бот запитує суму, категорію, дату та коментар.
   - Після кожного додавання витрати автоматично перевіряється баланс. Якщо витрати перевищують бюджет, виводиться попередження.
4. *Збереження*: Після будь-якої зміни (зміна бюджету або додавання витрати) дані автоматично зберігаються назад у JSON-файл.

#Структура даних 
Дані зберігаються у форматі JSON у вигляді словника:
json
{
  "budget": 1000.0,
  "expenses": [
    {
      "amount": 150.0,
      "category": "їжа",
      "date": "2026-03-05",
      "comment": "обід в університеті"
    }
  ]
}
